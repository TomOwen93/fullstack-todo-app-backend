My Todo API

RESTful JSON API

Retrieve all todos

GET /to-dos

Create a todo

POST /to-dos

Retrieve a specific todo

GET /to-dos/{id}

Update a specific todo

PATCH /to-dos/{id}

Attaching a JSON body with the possible below fields:

  • title (string)
  • message (string)

Delete a specific todo

DELETE /todos/{id}
